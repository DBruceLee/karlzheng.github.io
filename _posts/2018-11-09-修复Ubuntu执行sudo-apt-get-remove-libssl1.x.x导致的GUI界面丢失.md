## 修复Ubuntu执行sudo apt get remove libssl1.x.x导致的GUI界面丢失

Ubuntu执行sudo apt get remove libssl1.x.x导致的GUI界面丢失:

[https://askubuntu.com/questions/914234/sudo-apt-get-remove-libssl1-0-0-libssl-dev-by-mistake](https://askubuntu.com/questions/914234/sudo-apt-get-remove-libssl1-0-0-libssl-dev-by-mistake)

修复方法：

[https://blog.csdn.net/bupt073114/article/details/47416455](https://blog.csdn.net/bupt073114/article/details/47416455)

```shell
# 可先更新/etc/apt/source.list
sudo apt-get update
sudo apt-get install --reinstall ubuntu-desktop
sudo apt-get install unity
sudo shutdown -r now
```

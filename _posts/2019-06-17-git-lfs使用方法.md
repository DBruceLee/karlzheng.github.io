## git lfs使用方法

- **使用方法**

- 告诉lfs需要管理的大文件，比如model.pb，运行命令`git lfs track model.pb`
- 将管理文件.gitattributes添加入git仓库`git add .gitattributes`
- 将大文件添加入git仓库，然后和其它添加方式一样

```
git add model.pb
git commit -m "add model file"
git push
```
